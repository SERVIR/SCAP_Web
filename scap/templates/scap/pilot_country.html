Visualize forest cover changes around protected areas

This tool serves as two comparisons; (1) Comparing annual forest coverage using the same land use/land cover dataset to determine changes in forest (2) Comparing the same annual forest coverage but between two separate datasets to demonstrate where the datasets may diverge. When viewing the differences between two different years using the same dataset, changes will appear in red to represent potential deforestation. When viewing the differences between the same year but two different datasets, changes will appear in yellow to represent divergence between datasets.

Explore how different sources estimate emissions

This tool provides a large array of carbon emission estimations by displaying combinations of activity data (found by taking the difference between two annual forest cover maps) and above ground biomass measurements. Depending on the estimated hectares of activity data along with the associated biomass of the deforested land, users can compare across the ensemble of estimates to explore data trends across time and space. As not all forest cover datasets have full temporal coverage from 2000 - 2020, this chart will contain gaps in the trends lines to avoid false estimations in emissions.

Understanding the map

Forest cover analysis

Comparing different years for same forest cover collection

The left side shows the earlier year in bright green, indicating the places that were originally identified as forest.

The right side shows the later year in darker green, with the areas that have lost forest cover in red.

Comparing data from different forest cover collections

The left side shows the first dataset in green, while the yellow pixels indicate locations identified as forest only in the second dataset.

The right side shows the second dataset in green, with yellow pixels indicating locations identified as forest only in the first dataset
